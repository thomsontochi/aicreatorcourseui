# PromptEdit — UI/UX Redesign

A frontend redesign of **PromptEdit**, a marketplace for AI video prompts, DaVinci Resolve templates, and Premiere Pro assets. Built to demonstrate design sensibility, frontend craft, and the ability to translate a complex product vision into clean, production-ready code without a Figma handoff.

## The Problem

Marketplaces for creative assets often look like they were built by engineers who never talked to the people buying the assets. Cluttered layouts, unclear value propositions, no visual hierarchy. PromptEdit had the product — it needed the presentation to match.

## Design Decisions

- **Color system:** White surface, deep green for headlines and primary actions, muted gray body copy — professional without being cold
- **Mobile-first:** Every breakpoint designed and tested, not retrofitted
- **No framework overhead:** Pure HTML, Tailwind CSS, and vanilla JS — fast load, easy to hand off to any stack
- **Zero generic AI aesthetics:** Every component decision made to feel like a real product, not a template output
